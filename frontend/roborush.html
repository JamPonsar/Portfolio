Robo-Rush

2D Side Scroller Game with Advanced Mechanics

March 2023 - May 2023

View on GitHub Play Game

Project Overview

Robo-Rush is an engaging 2D side-scroller running game built from scratch using TypeScript and the Wolfie2D Game Engine. Developed in collaboration with two partners at Stony Brook University, this project showcases game development skills, creative design, and team collaboration in creating a fully functional gaming experience.

Game Features

Dynamic Gameplay

Fast-paced side-scrolling action with smooth character movement and responsive controls

Unique Levels

Multiple challenging levels with increasing difficulty and unique environmental obstacles

Double Jumping

Advanced platforming mechanics including double jump for enhanced player mobility

Dynamic Obstacles

Variety of moving and static obstacles requiring quick reflexes and strategic timing

Player Animations

Smooth, frame-based animations for running, jumping, and special actions

Scoring System

Competitive scoring mechanics with progress tracking and achievements

Technical Implementation

Game Development

  • TypeScript: Type-safe game logic and object-oriented design
  • Wolfie2D Engine: Custom game engine for 2D graphics and physics
  • Game Loop: Efficient update and render cycles for smooth gameplay
  • State Management: Game state handling for menus, gameplay, and transitions

Game Design & Art

  • Level Design: Creative level layouts with balanced difficulty progression
  • Digital Art: Custom artwork and sprite creation for game assets
  • Animation Systems: Frame-based animation for character movements
  • UI/UX Design: Intuitive game interface and menu systems

Backend & Data

  • Firebase: Cloud storage for game data and user progress
  • Real-time Database: Score tracking and leaderboard functionality
  • Authentication: User account management and progress saving

Advanced Game Mechanics

Physics System

Implemented realistic gravity, collision detection, and momentum-based movement for natural gameplay feel.

Procedural Elements

Dynamic obstacle placement and timing variations to keep gameplay fresh and challenging.

Power-ups & Abilities

Special abilities and temporary power-ups that enhance gameplay and provide strategic depth.

Progressive Difficulty

Intelligent difficulty scaling that adapts to player performance and maintains engagement.

Development Process & Collaboration

Concept & Planning

Collaborated with team members to brainstorm game concept, define core mechanics, and establish art direction. Created detailed game design documents and technical specifications.

Art & Design

Led the creative direction, designing character sprites, environment assets, and UI elements. Developed a cohesive visual style that complemented the fast-paced gameplay.

Programming & Implementation

Implemented core game mechanics, physics systems, and level progression using TypeScript and the Wolfie2D engine. Focused on optimizing performance for smooth 60fps gameplay.

Testing & Polish

Conducted extensive playtesting sessions to refine game balance, fix bugs, and enhance user experience. Iteratively improved gameplay mechanics based on feedback.

Challenges & Solutions

Performance Optimization

Maintaining consistent frame rates while handling multiple animated sprites and collision detection required careful optimization of the game loop and efficient memory management.

Team Coordination

Coordinating development tasks across three team members involved implementing version control workflows and establishing clear communication protocols for asset integration.

Game Balance

Achieving the right difficulty curve that challenged players without frustrating them required extensive playtesting and iterative design adjustments.
